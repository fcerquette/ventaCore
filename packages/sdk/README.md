# @base-template/sdk

SDK **autogenerado** a partir del Swagger JSON del API (`typescript-axios`).

> ⚠️ No edites `src/` a mano. Se regenera con:
>
> ```bash
> # con el API corriendo en http://localhost:3000
> npm run generate:sdk
> ```

El frontend importa el SDK vía el alias `@base-template/sdk`.
Hasta que generes el SDK por primera vez, `src/` está vacío (solo `.gitkeep`).
